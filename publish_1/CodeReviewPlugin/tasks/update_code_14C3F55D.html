Task: Update Code
Disciplines: Software development
Purpose
Address a feedback by making necessary changes
Relationships
RolesPrimary Performer:
  • Software Developer
Additional Performers:
  • Project Manager
  • Senior Reviewer
InputsMandatory:
  • Review
  • Source Code
    		Optional:
    • Analysis Report
    • Test report
    • Tests
      Outputs
      • Updated Code
        Main Description
        Update is based on the feedback of reviewer, tester, quality analyst or project manager and to make code clean, stable, etc.
        Steps
        Address reviewer feedback by making necessary changes to the code/architecture
        Respond to reviewer comments and questions to clarify any issues or not obvious parts of code that may help them to make better review
        *Test the updated code to ensure it still meets functional requirements and is free of bugs
        Document any changes made to the code and the reasons why they have been made
        *Submit the code for review again if needed
