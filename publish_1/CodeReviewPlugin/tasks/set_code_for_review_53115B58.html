      Main Description

      Contains necessary additions to code so it is easier to comment/write a review on. Also, may have specific context to clarify some features for reviewer so it is easier to write a feedback.

      Steps
      Prepare the code for review by checking if it follows coding standards, project guidelines and other requirements
      Create a code review request
      *Ensure that request can be seen in a system(i.e GitHub) or notify reviewers/testers or others who is responsible for reviews and provide them with access to the code and any relevant documentation
