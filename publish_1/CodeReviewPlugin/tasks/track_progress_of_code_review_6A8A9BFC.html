Task: Track Progress of Code Review
Disciplines: Project Management
Purpose
To manage and coordinate team members to ensure timely completeion of code review processes.
Relationships
Parent
  • Code review best practices
RolesPrimary Performer:
  • Project Manager
Additional Performers:
  • Senior Reviewer
InputsMandatory:
  • Project Review Plan
  • Review Documents
    		Optional:
    • Analysis Report
    • Code Review Plan
    • Specification for reviews
      Outputs
      • Project Status Reports
      • Project Task List
        Steps
        Monitor the progress of the code review, including the completion of tasks and provided feedback
        Ensure that the review process is going according to the established timeline
        *Make sure that any updates or changes in the review process are known by all participants of the process
