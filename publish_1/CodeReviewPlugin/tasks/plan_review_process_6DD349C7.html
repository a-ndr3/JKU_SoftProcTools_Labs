Task: Plan Review Process
Disciplines: Project Management
Purpose
Plan and schedule review process as well as assign reviewers and set deadlines
Relationships
RolesPrimary Performer:
  • Senior Reviewer
Additional Performers:
  • Project Manager
  • Reviewer
InputsMandatory:
  • Specification for reviews
    		Optional:
    • None
    Outputs
    • Code Review Plan
    • Project Review Plan
    • Project Task List
      Steps
      Define what should be reviewed
      Define who should participate in it (developers, testers, reviwers, quality analysts etc.)
      Create a schedule of code reviews, set deadlines and timeline for review process
      Inform all participants about the plan and help them to follow it
