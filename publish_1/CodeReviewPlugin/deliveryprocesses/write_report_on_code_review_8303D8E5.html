Outputs

Main Description
To monitor progress of code review. To make it easier for code reviewers to follow guidelines and complete tasks within timeline.
Steps
Summarize the key findings and recommendations from the code review process
Document any problems and issues that have happened
Write what can possibly be done to resolve these issues and make suggestions for possible improvements so to not have such problems in the future
Add information on the impact of the code review to the project
Share the report (for example with development team, project manager, etc.)
